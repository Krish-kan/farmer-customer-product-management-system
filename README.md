# Farmer-Customer Product Management System (DBMS)

## Overview
A relational database system enabling direct interaction between farmers and customers.
The system manages users, products, orders, transactions, and feedback using a normalized schema.

## Features
- Farmer and customer role-based users
- Product listing and inventory tracking
- Orders, carts, transactions, and feedback
- ER modeling and normalization (up to 3NF)
- Triggers for logging and integrity

## Tech Stack
Oracle SQL / MySQL, DBMS concepts

## Folder Structure
- sql/: SQL schema and queries
- docs/: Project report
- er-diagram/: ER diagrams
- screenshots/: SQL outputs
